# Hospital Appointment System

## Project Description
This is a Hospital Appointment System developed using Django. The system allows patients to register, doctors to be managed, and appointments to be booked online.

## Features
- Patient Registration
- Doctor Management
- Book Appointment
- View Patient List
- View Doctor List
- Appointment Management

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Project Status
Currently under development.
